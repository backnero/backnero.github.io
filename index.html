微信用户unionID获取策略调整

为保护用户隐私，用户unionID需用户主动授权或关注公众号后才可获得，微信公众号接口将做以下调整：

1、微信授权登录功能中的换取access_token接口

本接口在scope参数为snsapi_base时不再提供unionID字段。

调整建议：

只需将scope参数修改为snsapi_userinfo即可重新获得用户unionID。

详情可参考微信授权登录文档

2、获取用户基本信息接口

本接口在用户未关注公众号时，将不返回用户unionID信息。

调整建议：

已关注的用户，开发者可使用“获取用户基本信息接口”获取unionID；

未关注用户，开发者可使用“微信授权登录接口”并将scope参数设置为snsapi_userinfo，获取用户unionID

详情可参考获取用户基本信息接口文档

以上变更将在2017年12月31日之前逐步生效，请开发者尽快更新。​

git命令备忘

本地初始化git

设定用户名及邮箱

git config –global user.name “你的名字或昵称”
git config –global user.email “你的邮箱”

作为 git 的基础配置，作用是告诉 git 你是谁，你输入的信息将出现在你创建的提交中。

关联远程仓库

git remote add origin <你的项目地址>

//注:项目地址形式为:http://git.oschina.net/xxx/xxx.git或者 git@git.oschina.net:xxx/xxx.git

查看目录关联情况

git remote -v

删除已有git关联库

git remote rm origin

关联多远程库需要设置名称

git remote add github

git@github.com:user/project.git
增加一个名为github的库

获取代码

git clone

克隆远程代码

git pull origin master

获取master分支代码

上传代码

git add .

修改后加入暂存区

git commit -m “提交信息”

提交代码备注

git tag 标签名称

用于版本控制，如V1.0.1

git tag -a v1.0 -m ‘备注信息’

带备注的标签信息

git push origin master

上传代码到master分支

git push github master

git push gitee master

若本地目录关联多个远程库，可使用该命令上传到不同的远程库。

侠客行

李白（701～762），字太白，号青莲居士。是屈原之后最具个性特色、最伟大的浪漫主义诗人。有“诗仙”之美誉，与杜甫并称“李杜”。


赵客缦胡缨，吴钩霜雪明。银鞍照白马，飒沓如流星。

十步杀一人，千里不留行。事了拂衣去，深藏身与名。

闲过信陵饮，脱剑膝前横。将炙啖朱亥，持觞劝侯嬴。

三杯吐然诺，五岳倒为轻。眼花耳热后，意气素霓生。

救赵挥金锤，邯郸先震惊。千秋二壮士，烜赫大梁城。

纵死侠骨香，不惭世上英。谁能书阁下，白首太玄经。

通过百度地图接口实现最近商家展示

Route Matrix API v2.0 Beta是一套以HTTP/HTTPS形式提供的批量算路接口，返回路线规划距离和行驶时间。

业务需求

在客户端中获取用户定位信息，并与列表中商家坐标计算距离，商家列表按照距离由小到大排列。

业务分析

根据需求可知，

  1. 获取用户的坐标，设为[参数1]。获取商家列表设置为[数组1]。
  2. 将参数1与数组1中的商家坐标点计算距离，
  3. 返回按距离由小到大的[数组2]。

Route Matrix API v2.0 功能介绍

  • 批量计算路线的距离和耗时
  • 支持驾车、骑行和步行三种形式
  • 步行时任意起终点之间的距离不得超过200KM，超过此限制会返回参数错误
  • 一次最多计算50条路线，起终点个数之积不能超过50。 比如2个起点25个终点，50个起点1个终点等

实现思路

  1. 通过客户端获取用户定位信息。
  2. 将用户坐标传递给API获得省市区信息并传递给服务端。
  3. 根据省市区信息服务端筛选商家列表。
  4. 调用Route Matrix API v2.0 计算用户定位点与商家的距离。
  5. 服务端将计算结果按距离由小到大形成数组，通过json传递给客户端。
  6. 客户端展示附近商家列表。

请求接口地址

  • 驾车

    http://api.map.baidu.com/routematrix/v2/driving?

  • 骑行

    http://api.map.baidu.com/routematrix/v2/riding?

  • 步行

    http://api.map.baidu.com/routematrix/v2/walking?
